# Geraldo

Projeto para simplificar cotações de produtos para empresas e fornecedores

## Requirimentos

- Python 3.12
- Docker
- Node.js 20

## Uso

```shell
copy backend/env_sample backend/.env
# resolve .env config
docker-compose up -d
```
